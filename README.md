## コマンドライン引数でパラメータを指定する  
- デバイス(第一引数)  
  - デフォルト値  
    /dev/ttyS0  
  - 指定できる値  
    /dev/tty*  

- ボーレート(第二引数)  
  - デフォルト値  
    115200  
  - 指定できる値  
    9600, 38400, 115200  

- データ幅(第三引数)  
  - デフォルト値  
    8  
  - 指定できる値  
    7, 8  

- パリティ有無(第四引数)  
  - デフォルト値  
    none  
  - 指定できる値  
    odd, even  

- ストップビット幅(第五引数)  
  - デフォルト値  
    1  
  - 指定できる値  
    1, 1.5, 2
